## Mekanik Dasar Isi Ulang

Di Attack on Titan Revolution, bilah Anda tidak permanen. Setiap serangan terhadap tengkuk atau anggota tubuh Titan mengurangi daya tahan set bilah Anda saat ini. Setelah meteran daya tahan mencapai nol, bilah Anda akan hancur, atau menjadi "tumpul," yang secara signifikan mengurangi damage dan kecepatan Anda.

Untuk melakukan isi ulang, Anda harus memiliki bilah cadangan yang tersedia di sarung perlengkapan Omni-Directional Mobility (ODM) Anda. Secara default, input untuk tindakan ini adalah **tombol 'R'** pada keyboard Anda. Saat Anda menekan ini, karakter Anda akan membuang bilah saat ini dan memasukkan sepasang bilah baru ke dalam gagangnya. Animasi ini memakan waktu beberapa detik, di mana Anda tidak dapat menyerang, menjadikan pengaturan waktu sebagai faktor paling kritis dalam pertempuran.

| Tindakan | Tombol (PC) | Pengontrol (Xbox/PS) | Efek |
| :--- | :--- | :--- | :--- |
| **Isi Ulang Bilah** | **R** | **X / Kotak** | Membuang bilah tumpul untuk yang baru |
| **Cek Daya Tahan** | **UI Overlay** | **UI Overlay** | Memantau sisa kesehatan bilah |
| **Tukar Cepat** | **Otomatis** | **Otomatis** | Dipicu oleh perk tertentu |

> **⚠️ Peringatan:** Jangan menunggu sampai bilah Anda benar-benar patah untuk mengisi ulang. Bilah yang tumpul memiliki penalti damage yang sangat besar, sering kali membutuhkan serangan tiga kali lebih banyak untuk mendapatkan kill.

## Skill dan Perk untuk Isi Ulang yang Efisien

Untuk mengoptimalkan proses **Attack on Titan Revolution cara isi ulang pedang**, Anda dapat berinvestasi dalam skill dan perk tertentu dalam sistem progres game. Peningkatan ini dapat mengurangi waktu yang dibutuhkan untuk menukar bilah atau bahkan memberikan kesempatan untuk tidak mengonsumsi set bilah saat mengisi ulang.

1. **Handy Perk**: Ini adalah keharusan bagi pemain yang merasa animasi isi ulang terlalu lambat. Ini mengurangi waktu isi ulang dengan persentase yang signifikan, memungkinkan Anda kembali beraksi lebih cepat.

3. **Blade Conservation**: Skill pasif ini memberi Anda peluang 10-15% untuk mempertahankan set bilah saat ini bahkan ketika daya tahan mencapai nol, secara efektif memberi Anda isi ulang "gratis".

> **💡 Tip:** Jika Anda bermain dalam skuad, koordinasikan isi ulang Anda. Jangan biarkan seluruh tim mengisi ulang pada saat yang sama, karena ini membuat kelompok tidak berdaya terhadap serbuan Titan.

## FAQ

**Q: Mengapa saya tidak bisa mengisi ulang pedang saya di Attack on Titan Revolution?**

A: Ada dua alasan utama mengapa Anda mungkin tidak bisa mengisi ulang. Pertama, periksa apakah Anda masih memiliki set bilah cadangan di inventaris Anda; jika bilah habis, tombol 'R' tidak akan berfungsi. Kedua, pastikan Anda tidak sedang berada di tengah animasi lain, seperti saat ditangkap oleh Titan atau menggunakan skill khusus.

**Q: Bagaimana cara mendapatkan lebih banyak bilah cadangan untuk perlengkapan saya?**

A: Anda dapat meningkatkan kapasitas bilah cadangan dengan meng-upgrade sarung bilah (sheaths) di Blacksmith atau dengan melengkapi perlengkapan ODM tingkat yang lebih tinggi. Beberapa perk tertentu di pohon keterampilan juga memungkinkan Anda membawa set ekstra.

**Q: Apakah isi ulang pedang saya membutuhkan uang atau emas?**

A: Tidak, tindakan mengisi ulang selama misi tidak memerlukan biaya mata uang. Namun, Anda harus "membeli" atau melengkapi set bilah sebagai bagian dari loadout Anda sebelum misi dimulai. Mengisi ulang bilah di peti suplai selama pertandingan juga gratis.

**Q: Apa cara tercepat untuk menguasai Attack on Titan Revolution cara isi ulang pedang?**

A: Cara terbaik untuk menguasai isi ulang adalah dengan melatih "Air Reloading." Berlatihlah menggunakan grapple pada bangunan, melompat, dan menekan 'R' saat berada di udara. Ini membuat Anda tetap lincah dan memastikan pada saat Anda mendarat atau mencapai Titan berikutnya, bilah Anda sudah baru dan siap untuk bertempur.
